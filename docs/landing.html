A minimal, self-hosted paste service for the command line. No database. No frameworks. Just files.

bash
$ echo "hello world" | naste
https://paste.semi.sh/abc123
$ naste deploy.sh -s deploy
https://paste.semi.sh/deploy
$ cat secrets.env | naste -p -s secrets
https://paste.semi.sh/private/secrets
$

Install

# Install via go
go install github.com/semi710/nastebin/cmd/naste@latest

# Or run with nix (no install needed)
echo "hello world" | nix run github:semi710/nastebin#naste --

Usage

# Pipe from stdin
echo "hello world" | naste
cat file.go | naste

# File argument (auto-detects language)
naste main.go
naste deploy.sh
naste config.yaml

# Custom slug
naste file.txt -s myfile

# Private paste
naste secrets.env -p -s secrets

# Force overwrite
naste updated.go -f -s mycode

# Specify language
cat code.txt | naste -l go

Features

⚡

Zero Database

Filesystem-based storage. No PostgreSQL, no MongoDB, no Redis.

🔒

Private Pastes

HTTP Basic Auth for sensitive content. No accounts needed.

🏷️

Custom Slugs

Readable URLs like /deploy-script instead of random IDs.

🎨

Syntax Highlighting

Auto-detects language from file extension. Dark theme in browser.

📦

Single Binary

One Go binary. Deploy anywhere in seconds.

🐳

Docker Ready

Multi-arch images (amd64 + arm64) via GitHub Actions.

Configuration

VariableDefaultDescription
PORT8080HTTP listen port
DATA_DIR/data/pasteStorage directory
PRIVATE_USER(empty)Username for private pastes
PRIVATE_PASS(empty)Password for private pastes

Create ~/.config/naste/config.toml for CLI defaults:

endpoint = "https://paste.semi.sh"
user = "admin"
password = "secret"

Deploy

Docker (recommended):

docker run -d \\
  --name naste \\
  -p 8080:8080 \\
  -e PRIVATE_USER=admin \\
  -e PRIVATE_PASS=secret \\
  -v /var/lib/naste/data:/data/paste \\
  ghcr.io/semi710/naste-server:latest

Nix (one-liner deploy):

# Deploy locally
nix run github:semi710/nastebin#deploy

# Deploy to remote server
nix run github:semi710/nastebin#deploy -- user@server

Tailscale (no domain needed):

sudo tailscale funnel 8080
# → https://your-machine.tailnet-name.ts.net

API

# Create paste
POST /api/paste
Headers: X-Slug, X-Private, X-Lang
Body: raw text

# Retrieve paste
GET /{slug}           # Browser: HTML with syntax highlighting
GET /{slug}?raw=1     # Raw text

# Private paste
GET /private/{slug}   # Requires Basic Auth

# Health check
GET /health
